# 🚀 EntreSkill Hub

> **Skill-to-Startup Enablement Platform**

EntreSkill Hub is a full-stack MERN web application that helps aspiring entrepreneurs transform their skills into successful startup opportunities through personalized business ideas, expert mentorship, structured learning resources, and startup roadmaps.

---

# 📖 Project Overview

EntreSkill Hub bridges the gap between skills and entrepreneurship by providing a centralized platform where users can:

- Discover startup ideas based on their interests.
- Learn through curated resources.
- Connect with experienced mentors.
- Track startup progress using personalized roadmaps.
- Build entrepreneurial knowledge through practical guidance.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/H99ik/EntreSkill-Hub.git
```

---

## Frontend Setup

```bash
cd client

npm install

npm run dev
```

---

## Backend Setup

```bash
cd server

npm install

npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

EMAIL_USER=your_email

EMAIL_PASS=your_email_password
```
